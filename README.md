## 📌 Overview  
This repository contains **machine learning classification projects** built using Python and Scikit‑learn. Each project demonstrates end‑to‑end ML workflows—from data preprocessing to model evaluation.  
Current highlight: **Credit Card Fraud Detection** using supervised learning techniques.

---

## 📂 Project Files  
- `Classification_Project_3_(Credit_Card_Fraud_Detection).ipynb` – Notebook with data cleaning, feature engineering, model training, and evaluation.  
- `train.csv` – Dataset used for training and testing models.  

---

## 🎯 Objectives  
- Detect fraudulent transactions using classification algorithms.  
- Apply **data preprocessing, scaling, and feature selection**.  
- Compare models such as **Logistic Regression, Random Forest, and XGBoost**.  
- Evaluate performance using **accuracy, precision, recall, and F1‑score**.

## 📊 Key Insights  
- Fraud detection requires handling **imbalanced datasets**.  
- **Random Forest** and **XGBoost** often outperform linear models in precision.  
- Feature scaling and proper validation significantly improve accuracy.
